# Workday Hourly Planner


## Purpose

To create an hourly planner for the workday, where the entries for the hours persist across refresh, and the time slots are color coded to past, present and future. 

## Built With
 - HTML
 - CSS
 - JavaScript
 - JQuery
 - Moment
 - Bootstrap
